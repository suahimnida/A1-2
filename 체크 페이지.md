인터넷 정보를 받아와서 여행지 추천해주는 파이썬 프로그램 만들기

- 여러 API를 엮은 인사이트 제작
- Python에서 API 호출 -> LLM API가 여행지 추천 -> 지도 API가 위치 추적 -> 최종 리포트 생성
- LLM API와 지도 API를 조합하여 국내 여행 추천 프로그램 개발
- 사용자가 여행 날짜 입력 -> LLM 해당 시기에 여행하기 좋은 지역 추천 -> 지도 API로 맛집 정보 검색 -> 최종 여행 리포트 생성
- 단순 하나의 API를 호출하는 것을 넘어, 여러 API의 데이터를 조합하고 AI로 인사이트를 도출하는 프로그램 제작

**과제 목표**
REST API의 요청/응답 구조와 HTTP 메서드(GET/POST)의 차이를 말로 설명
LLM 출력 결과를 구조화(JSON)하여 다음 단계(지도/장소 검색)의 입력으로 활용하는 흐름 설명
외부 API 호출에서 발생하는 대표 오류(인증/쿼터/네트워크/파싱)와 대응 원칙 설명
API 키를 코드에서 직접 작성하지 않고 .env/환경변수로 관리하는 이유 설명

1. CLI 기반 Python 프로그램

입력/요청: -date "YYYY-MM-DD" (필수)
출력/화면: 진행 로그 + 결과 저장 경로 안내

2. 실행 결과 데이터

출력/화면: results/ 폴더에 아래 파일 생성
원본 데이터 JSON 1개 이상 (1차 추천 결과 + 맛집 검색 결과 포함)
최종 여행 리포트 markdown 1개

3. README.md

프로그램 개요, 실행 방법, API 키 설정 방법, 결과물 확인 방법
API 키가 유출되지 않도록 주의 사항 포함
