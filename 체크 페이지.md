인터넷 정보를 받아와서 여행지 추천해주는 파이썬 프로그램 만들기

- 여러 API를 엮은 인사이트 제작
- Python에서 API 호출 -> LLM API가 여행지 추천 -> 지도 API가 위치 추적 -> 최종 리포트 생성
- LLM API와 지도 API를 조합하여 국내 여행 추천 프로그램 개발
- 사용자가 여행 날짜 입력 -> LLM 해당 시기에 여행하기 좋은 지역 추천 -> 지도 API로 맛집 정보 검색 -> 최종 여행 리포트 생성
- 단순 하나의 API를 호출하는 것을 넘어, 여러 API의 데이터를 조합하고 AI로 인사이트를 도출하는 프로그램 제작
